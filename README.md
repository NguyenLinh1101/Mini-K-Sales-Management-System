# Mini K Sales Management System

## Overview

This project was developed as part of the Project Management course.

The project focuses on proposing and planning a Sales Management System for Mini K, a supermarket chain with 15 stores located across different provinces and cities in Vietnam.

Instead of developing the software itself, the project concentrated on project initiation, planning, procurement preparation, and monitoring activities that would be required before implementation.

---

## Project Objective

The proposed system aims to support:

- Centralized sales management across all stores
- Inventory monitoring and control
- Procurement management
- Operational reporting
- Better coordination between headquarters and branch stores
